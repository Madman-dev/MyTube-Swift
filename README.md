# MyTube-Swift
- **유튜브 어플 클론 프로젝트.**
- **영상 플랫폼의 중심이 된 유튜브.**
전세계 모두에게 동일한 퀄리티의 콘텐츠를 제공하는 기업인만큼,
동일한 구현을 위해 구성하되 조금은 더 깔끔한 UI를 구현할 수 있도록 정리
- **구글 API를 통해 유튜브 영상을 크롤링, 메인 페이지에 영상 출력.**
마이 페이지에 시청 영상과 구독하는 유튜버들을 표시할 수 있도록 구현
